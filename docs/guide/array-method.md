# Array 方法总结

方法名 | 功能 | 参数 | 返回值 | 改变原数组 | 备注
--- | --- | --- | --- | :---: | --- 
`Array.isArray()` | 方法返回一个布尔值,表示参数是否为数组 | 被判断的数组 | 布尔值 | 否 | 数组的静态方法 
 | | | | |
`toString()` | 返回数组的字符串形式,以逗号分隔 | - | 字符串 | 否 | -
`join()` | 指定分隔符,将所有数组成员连接为一个字符串返回.如果不提供参数,默认用逗号分隔 | 分隔符 | 字符串 | 否 | `undefined`、`null`、空位,会被转成空字符串
 | | | | |
`push()` | 在数组的末端添加一个或多个元素,返回添加新元素后的数组长度 | 添加的元素, ... | 数组长度 | 是 | -
`pop()` | 删除数组的最后一个元素,并返回该元素 | - | 删除的元素 | 是 | 对空数组使用,不会报错,返回`undefined`
`unshift()` | 在数组的第一个位置添加一个或多个元素,返回添加新元素后的数组长度 | 添加的元素, ... | 数组长度 | 是 | -
`shift()` | 删除数组的第一个元素,并返回该元素 | - | 删除的元素 | 是 | -
 | | | | |
`reverse()` | 颠倒排列数组元素,返回改变后的数组 | - | 改变后的数组 | 是 | -
`sort()` | 对数组成员进行排序,默认是按照字典顺序排序 | 函数 | 改变后的数组 | 是 | 1.`sort`默认方法不是按照大小排序,而是按照字典顺序.数值会被先转成字符串,再按照字典顺序进行比较;2.`sort`的参数函数的返回值>0:第一个成员排在第二个成员后面;其他情况,都是第一个元素排在第二个元素前面
 | | | | |
`conact()` | 多个数组的合并,将新数组的成员,添加到原数组成员的后部,返回一个新数组 | 数组 | 返回新数组 | 否 | `var newArray = oldArray.concat();` 实现浅拷贝
`slice()` | 提取目标数组的一部分,返回一个新数组 | 起始位置(从0开始),终止位置 注:[)区间 | 返回新数组 | 否 | 1.参数是负数,则表示倒数计算的位置;2.忽略终止位置,则表示直到数组结尾;3.如果第一个参数大于等于数组长度,或者第二个参数小于第一个参数,则返回空数组;3.`var newArray = oldArray.slice();` 实现浅拷贝;4.`slice`方法的重要应用,是将类似数组的对象转为真正的数组`Array.prototype.slice.call({ 0: 'a', 1: 'b', length: 2 })`
`splice()` | 删除原数组的一部分成员,并可以在删除的位置添加新的数组成员,返回值是被删除的元素 | start(起始位置), count(个数), addElement1, ... | 被删除的元素 | 是 | 1.参数是负数,则表示倒数计算的位置;2.忽略终止位置,则表示直到数组结尾;3.单纯地插入元素,`splice`方法的第二个参数可以设为0
 | | | | |
`map()` | 将数组的所有成员依次传入参数函数,然后把每一次的执行结果组成一个新数组返回 | 操作函数(elem, index, arr),绑定的this值 | 执行后的新数组 | 否 | `map`方法不会跳过`undefined`和`null`,但是会跳过空位
`forEach()` | 对数组的所有成员依次执行参数函数,无返回值,只操作数据 | 操作函数(elem, index, arr),绑定的this值 | - | 否 | `forEach`方法无法中断执行,总是会将所有成员遍历完
`filter` | 过滤数组成员,满足条件的成员组成一个新数组返回 | 操作函数(elem, index, arr),绑定的this值 | 满足条件的成员组成的新数组 | 否 | 所有数组成员依次执行该函数,返回结果为true的成员组成一个新数组返回
`some()` | 只要一个成员的返回值是`true`,则整个`some`方法的返回值就是`true`,否则返回`false` | 操作函数(elem, index, arr),绑定的this值 | 布尔值 | 否 | 对于空数组,`some`方法返回`false`
`every()` | 所有成员的返回值都是`true`,整个`every`方法才返回`true`,否则返回`false` | 操作函数(elem, index, arr),绑定的this值 | 布尔值 | 否 | 对于空数组,`every`方法返回`true`
 | | | | |
`indexOf()` | 返回给定元素在数组中第一次出现的位置,如果没有出现则返回-1 | 要查找的元素, 从该位置开始向后匹配 | 位置 | 否 | 不能用来搜索`NaN`的位置,方法内部,使用===进行比较,而`NaN`是唯一一个不等于自身的值
`lastIndexOf()` | 返回给定元素在数组中最后一次出现的位置,如果没有出现则返回-1 | 要查找的元素, 从该位置起向前匹配 | 位置 | 否 | 不能用来搜索`NaN`的位置,方法内部,使用===进行比较,而`NaN`是唯一一个不等于自身的值
