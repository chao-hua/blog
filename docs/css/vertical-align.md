### 2.1 复选框、单选框与文字对齐问题

在字体 12px 情况下，主流浏览器都存在复选框、单选框与文字对不齐的问题。可以通过给复选框、单选框设置 `vertical-align` 来解决：

- `vertical-align: middle;`
- `vertical-align: bottom;`
- `vertical-align: -3px;`
