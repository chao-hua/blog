# 三亚攻略
## 景点

- 天涯海角
- 亚龙湾
    + 星级酒店，私人海滩，放松
    + 亚龙湾热带天堂森林公园 一般
- 蜈支洲岛
    + 乘船上岛差不多二十几分钟
    + 贵
    + 情人岛
- 大东海：热闹
- 小东海：人少，海水很浅
- 三亚湾
    + 人多
    + 往西走，人少
- 鹿回头山顶公园
    + 淘宝上把门票都买好 加上电瓶车才38块钱
    + 看到三亚的海岸线
- 南山
    + 南海观音
